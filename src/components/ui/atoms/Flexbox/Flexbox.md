Flexbox example:
```jsx
  <Flexbox type='div'>Default</Flexbox>
```

```jsx
  <Flexbox
    type='header'
  >
    This is an header
  </Flexbox>
```


```jsx
  <Flexbox
    hasBottomBorder
  >
    I have a bottom border
  </Flexbox>
```

```jsx
  <Flexbox
    justifyContent='space-between'
  >
    <p>I am right</p>
    <p>I am left</p>
  </Flexbox>
```
